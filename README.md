переход от бота к боту
Откройте командную строку (терминал).
   2. Выполните команду, чтобы найти запущенные процессы бота (они
      называются node.exe):

tasklist | findstr "node.exe"

Чтобы принудительно завершить их все одной командой, я сейчас
  выполню следующую команду:

  taskkill /F /IM node.exe /T

   * /F — принудительно завершить процесс.
   * /IM node.exe — найти все процессы с именем node.exe.
   * /T — завершить также и дочерние процессы.


pm2 start ecosystem.config.js
